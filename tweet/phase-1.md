## POST 2 — "AI wrote the code, not the product"

**Type:** [SINGLE]
**Post when:** 2–3 days after Thread 1
**Purpose:** Reframe the "AI built it" narrative. Draws replies and debates.

```
An AI wrote 100% of my app's code.

But an AI didn't:
→ Decide what to track
→ Know when the UI felt wrong
→ Figure out I'd quit if the input bar wasn't one-thumb reachable

"AI built it" and "AI made product decisions" are not the same thing.
```
*~258 chars*
